## 🚦 Getting Started

### Prerequisites
• Node.js & npm  
• MongoDB

### Installation

```sh
# Clone the repository
git clone https://github.com/lohanivaibhav4/Litpost.git

# Install dependencies
cd Litpost/Backend
npm install
cd ../Frontend
npm install

# Start the development servers
cd ../Backend
npm start
cd ../Frontend
npm run dev
```
